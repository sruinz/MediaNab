# MediaNab Support

For help with MediaNab, use the public GitHub issue tracker:

- Issues: https://github.com/sruinz/MediaNab/issues
- Repository: https://github.com/sruinz/MediaNab

## Before Opening an Issue

Please include:

- Browser and version, such as Microsoft Edge or Chrome.
- Operating system, such as macOS or Windows.
- MediaNab version, such as `1.2.4.06`.
- Whether the optional Companion is installed and connected.
- A short description of the page type, without sharing private URLs or account tokens.
- Relevant popup log messages, with personal paths, cookies, and account details removed.

## Scope

MediaNab can help with extension behavior, Companion setup, supported media detection, queue handling, and local save issues.

MediaNab cannot help bypass DRM, site access controls, paid-content restrictions, or third-party platform rules.
